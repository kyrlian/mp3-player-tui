# mp3-player-tui

TUI mp3 player in [python](https://www.python.org/), [textual](https://textual.textualize.io/), using [pygame.mixer.music](https://www.pygame.org/docs/ref/music.html)

## Install

```sh
poetry install
```

## Run

```sh
poetry shell
mp3player
```
